Instantly change admin email and any user's username without email confirmation or database edits.

== Description ==
This plugin helps site admins quickly:
* Change the WordPress admin email without waiting for confirmation
* Change any user's username directly from the dashboard

== Installation ==
1. Upload plugin zip via Plugins > Add New > Upload
2. Activate the plugin
3. Navigate to Settings > Quick Identity Change
